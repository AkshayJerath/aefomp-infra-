# AEFOMP Infrastructure as Code

## Overview

Infrastructure as Code repository for the Aircraft Engineering & Fleet Operations Management Platform (AEFOMP).

## Provisioned Resources

- Amazon EKS Cluster
- Amazon RDS PostgreSQL
- Amazon S3 Document Storage

## Environment

Production

## Cloud Provider

Amazon Web Services (AWS)

## Deployment Tool

Terraform

## Deployment Command

```bash
terraform init
terraform plan
terraform apply
```
